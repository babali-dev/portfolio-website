# Personal Portfolio Website

A modern, dark-themed personal portfolio website built using **HTML, CSS, and JavaScript**.

## Features
- Responsive layout
- Dark theme UI
- Smooth animations
- Mobile-friendly navigation
- Contact form (opens default email app with pre-filled message)

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Git & GitHub
- Netlify (for deployment)

## How to run locally
Open `index.html` in your browser.
